# SEMDS - 自进化元开发系统

## C4 模型架构文档

C4 模型由 Simon Brown 提出，用于描述软件架构的四个层次：
- **C1 - 系统上下文图**: 系统与外部用户/系统的交互
- **C2 - 容器图**: 系统内部的技术组件
- **C3 - 组件图**: 容器内部的模块划分
- **C4 - 代码图**: 类的关系（通过代码本身展现）

---

### 容器定义

| 容器 | 技术 | 职责 |
|------|------|------|
| Web 前端 | HTML + 内联 JS/CSS | 提供监控界面，实时展示进化进度 |
| FastAPI 服务 | Python + FastAPI | 提供 REST API 和 WebSocket 服务 |
| 进化引擎 | Python | 核心进化逻辑，协调各组件工作 |
| 数据层 | SQLite + Git | 持久化存储任务、进化历史、代码版本 |
| Docker 沙盒 | Docker | 隔离执行用户代码，防止恶意代码影响宿主 |

### 核心组件定义

| 组件 | 职责 | 输入 | 输出 |
|------|------|------|------|
| CodeGenerator | 调用 LLM 生成代码 | 任务描述、函数签名、前代代码 | 生成的 Python 代码 |
| TestRunner | 在沙盒中执行测试 | 代码、测试文件 | 测试报告（通过率、日志） |
| DualEvaluator | 双轨评分 | 测试报告、代码 | 综合评分、Goodhart 标记 |
| StrategyOptimizer | 选择进化策略 | 历史策略表现 | 下一轮策略配置 |
| TerminationChecker | 检查终止条件 | 当前代数、得分 | 是否继续/终止 |
| GitManager | 版本控制 | 代码、代数 | Git commit hash |

## API 架构

### REST API

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         FastAPI 路由层                                  │
│                                                                         │
│  POST   /api/tasks                    创建新任务                        │
│  GET    /api/tasks                    列出所有任务                      │
│  GET    /api/tasks/{id}               获取任务详情                      │
│  DELETE /api/tasks/{id}               删除任务                          │
│                                                                         │
│  POST   /api/tasks/{id}/start         启动进化                          │
│  POST   /api/tasks/{id}/pause         暂停进化                          │
│  POST   /api/tasks/{id}/resume        恢复进化                          │
│  POST   /api/tasks/{id}/abort         中止进化                          │
│                                                                         │
│  GET    /api/tasks/{id}/generations   获取进化历史                      │
│  GET    /api/tasks/{id}/best          获取最佳实现                      │
│  POST   /api/tasks/{id}/rollback/{gen} 回滚到指定代                      │
│                                                                         │
│  GET    /api/stats                    系统统计                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### WebSocket

```
WS /ws/tasks/{task_id}

消息格式:
{
    "type": "progress",

    "best_score": 0.91,
    "best_gen": 9,
```
