## 🎯 快速参考

### AI开发工作流程

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 需求解析 (15%)                                           │
│    - 阅读规格文档                                            │
│    - 评估理解度                                              │
│    - 识别风险和依赖                                          │
├─────────────────────────────────────────────────────────────┤
│ 2. 测试先行 (25%)                                           │
│    - 写测试（必须先失败）                                     │
│    - 验证测试覆盖                                            │
├─────────────────────────────────────────────────────────────┤
│ 3. 最小实现 (35%)                                           │
│    - 使测试通过的最少代码                                     │
│    - 即时质量门禁                                            │
├─────────────────────────────────────────────────────────────┤
│ 4. 重构优化 (15%)                                           │
│    - 消除重复                                                │
│    - 优化命名                                                │
│    - 保持测试通过                                            │
├─────────────────────────────────────────────────────────────┤
│ 5. 文档同步 (10%)                                           │
│    - 更新docstring                                           │
│    - 更新相关文档                                            │
│    - 验证一致性                                              │
└─────────────────────────────────────────────────────────────┘
```

### 质量门禁

```bash
# 代码提交前必须运行：
make check

# 包含：
# 1. black --check .          (代码格式)
# 2. isort --check-only .     (导入排序)
# 3. mypy core/ evolution/    (类型检查)
# 4. flake8                   (静态检查)
# 5. pytest --cov=...         (测试+覆盖率)
```

### 审查层级

| 层级 | 检查项                     | 自动化      |
| ---- | -------------------------- | ----------- |
| L1   | 基础合规（格式/类型/测试） | ✅ 全自动   |
| L2   | 代码规范（命名/注释/结构） | ⚠️ 半自动   |
| L3   | 测试质量（覆盖/独立/有效） | ⚠️ 半自动   |
| L4   | 安全性（注入/密钥/沙盒）   | ✅ 自动扫描 |
| L5   | 性能（复杂度/资源）        | ⚠️ 人工     |
| L6   | 可维护性（重复/圈复杂度）  | ⚠️ 半自动   |
| L7   | 文档同步                   | ⚠️ 人工     |

## 🚫 AI禁止清单

### 绝对禁止

- ❌ 虚报进度（未通过验收报告"完成"）
- ❌ 修改测试使测试通过
- ❌ 创建V2/增强版/改进版文件
- ❌ 跳过TDD流程
- ❌ 硬编码密钥/密码
- ❌ 使用eval/exec
- ❌ 无验证的用户输入

### 强烈不推荐

- ⚠️ 一个函数超过50行
- ⚠️ 圈复杂度超过10
- ⚠️ 重复代码（复制粘贴>3行）
- ⚠️ 测试覆盖率<80%
- ⚠️ 无类型注解的公共API
